Briefbox - Cosmos

This project is for a fake brief from Briefbox for an oil and candle company named Cosmos. The goal of the project was to create a set of 3 candle labels that have an elegant, yet down-to-earth feeling. Since the brief requested a hand-painted look, I simulated watercolors using Clip Studio Paint brushes. I based the colors off the ingredients for each label, and used the name of the blend to come up with the imagery. The background and text were combined in Illustrator to be adjusted. Finally, I used photoshop to create the mockups.
